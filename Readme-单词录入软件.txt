单词录入软件V1.0
制作者：岳阳市第十五中学 423班 梁家诺
--------------------------------------------------------------
Ⅰ.写在开始
      灵感来自B站上大学生”轻松高效使用Windows 10“中对单词的录入以及自己在用电脑积累单词时所发现的一个一个单词录入的麻烦（一个晚上只录入了150个单词），所以突发奇想做了这样一个软件。
       制作的时间是9月30日到10月17日，因为之前学习的是C++，所以Python可以算是从0开始学习。这一次编程时问题主要在库的使用上。国庆几天学习了Tkinter的编程教程，对于索引开始为0、制表符和表格的区别和大小写的严格非常烦。但是，最后还是坚持了下来，完成了对软件的编写。这一次的编程是由框架到区块写的，其中让我印象最深的就是Excel前面总是空很多排。但经过几个晚上，终于在计数器方面给它修复了。
       All in all,希望这个软件可以减轻同学们的压力，提高效率！
--------------------------------------------------------------
Ⅱ.文件清单
①主文件：单词录入.exe
②不能删除的文件：1.ico、英汉汉英词典.csv、numfile.txt
③可以替换的软件：words.xls
--------------------------------------------------------------
Ⅲ.注意事项：
1.“英汉汉英词典”为自制的本地词典，请务必保留
2.单词录入的默认格式名为"words.xls",请使用这个文件，因为清除后会出现错误
（或者新建一个文件（文件名、扩展名需要相同），然后在菜单中选择工具-删除计数器，计数器和excel将会自动重置）
3.务必保留numfile.txt
--------------------------------------------------------------

（1）编程语言为Python，GUI使用tkinter（和messagebox）制作，以及其它导入的库：csv（csv文件读取）、os（删除文件）、xlrd,xlwt（Excel的读取与写入）、xlutils.copy（Excel文件的追加写入）
（1）以上2、3点将会在以后的版本修复
